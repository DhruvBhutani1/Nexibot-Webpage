Introduction

NexiBot is a name that combines "Nexi," derived from the word "nexus," which means a connection or series of connections linking two or more things, with "Bot," short for robot. The name conveys a sense of futuristic technology and connectivity, suggesting that this bot is advanced, efficient, and central to the interactions within your Discord server.

About this Bot

The bot is both efficient and effective. The bot is hosted 24/7 and you can be reassured about it's uptime.

Technology

The bot is bulit with python using the discord.py library, same with the API the bot uses.

The bot uses it's own API to filter. With the help of our super fast and flexible API we are able to provide you the best and fastest features.

Commands

Syntax:
  • / Slash Commands

Music

  • Playing A Song
    • Description : Let's you play any music(on youtube) with the help of yt_dlp module.
    Usage : /play_song < A Url From Youtube>
    Example : /play_song https://youtu.be/uGcsIdGOuZY?si=WZJgb4qaC-y7E2JE
    Note : The bot will only play if the url is from youtube and you need to be in a voice channel to make it work.
  • Pausing The Song
    • Description : Let's you pause the song playing.
    Usage : /pause
    Example : /pause
    Note : You need to play a song to stop it.
  • Resumes the Song
    • Description : Let's you resumes the song.
    Usage : /resume
    Example : /resume
    Note : You need to stop a song to resume it.
  • Stops the Song
    • Description : Let's you stop the playing song.
    Usage : /stop
    Example : /stop
    Note :There should be a song playing to stop it.
